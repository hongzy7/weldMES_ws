# 华沿机器人智能焊接助手

面向焊接机器人现场作业的 MES 助手原型，用于展示机器人侧任务执行、数据采集、焊缝追溯和 BI 分析闭环。

## 项目结构

```text
weldMES_ws/
├─ src/
│  └─ index.html                         # 可点击 HTML 原型
├─ data/
│  └─ 华沿机器人智能焊接助手_数据表头说明.xlsx  # 系统读取/采集/记录字段说明
├─ docs/
│  └─ 焊接机器人MES助手.docx              # MES + BI 系统说明文档
├─ README.md
└─ .gitignore
```

## 使用方式

直接用浏览器打开 `src/index.html`，即可查看和点击操作原型。

## 当前内容

- 操作人员、检查员、班组长三类角色入口
- 焊前自检、焊材录入、工件确认、任务确认、工艺确认、焊接执行、焊缝检查、追溯和 BI 看板
- Excel 数据字典，包含待执行焊缝任务、操作人员信息、焊前准备、工艺参数、焊接过程、工件材料追溯、焊缝检查和 BI 指标输出字段
